Home

GeneralizedSasakiNakamura.jl computes solutions to the frequency-domain radial Teukolsky equation with the Generalized Sasaki-Nakamura (GSN) formalism.

The angular Teukolsky equation is solved with another julia package SpinWeightedSpheroidalHarmonics.jl using a spectral decomposition method.

Installation

To install the package using the Julia package manager, simply type the following in the Julia REPL:

using Pkg
Pkg.add("GeneralizedSasakiNakamura")

Note: There is no need to install SpinWeightedSpheroidalHarmonics.jl manually as it should be automatically installed by the package manager.

Easy to use

The following code snippet lets you solve the (source-free) Teukolsky function (in frequency domain) for the mode $s=-2, \ell=2, m=2, a=0.7, \omega=0.5$ that satisfies the purely-ingoing boundary condition at the horizon:

julia> using GeneralizedSasakiNakamura # This is going to take some time to pre-compile, mostly due to DifferentialEquations.jl
julia> # Specify which mode and what boundary condition
       s=-2; l=2; m=2; a=0.7; omega=0.5; bc=IN;
julia> # Specify where to match to ansatzes
       rsin=-20; rsout=250;
julia> # NOTE: julia uses 'just-ahead-of-time' compilation. Calling this the first time in each session will take some time
       R = Teukolsky_radial(s, l, m, a, omega, bc, rsin, rsout)TeukolskyRadialFunction(
    mode=Mode(s=-2, l=2, m=2, a=0.7, omega=0.5, lambda=1.6966094016353415),
    boundary_condition=IN,
    transmission_amplitude=1.0 + 0.0im,
    incidence_amplitude=6.536587661197792 - 4.941203897068912im,
    reflection_amplitude=-0.12824661912958304 - 0.4404813349665795im,
    normalization_convention=UNIT_TEUKOLSKY_TRANS
)

That's it!
